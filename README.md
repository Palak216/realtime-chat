# 💬 Pulse - Real-Time Chat Application

A modern real-time chat application built using the MERN Stack and Socket.IO.

---

## 🚀 Live Demo

Frontend:
https://realtime-chat-gizb-eight.vercel.app

Backend API:
https://pulse-chat-backend-deh0.onrender.com

---

## ✨ Features

- 🔐 JWT Authentication
- 💬 Real-time One-to-One Messaging
- 🟢 Online / Offline Status
- 📩 Instant Message Delivery
- 🔔 Unread Message Badge
- 🔍 Search Users
- 📱 Responsive UI
- ⚡ Socket.IO Integration
- ☁️ MongoDB Atlas Database
- 🚀 Vercel + Render Deployment

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Context API
- Axios
- Socket.IO Client

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- Bcrypt
- Socket.IO


## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Palak216/realtime-chat.git
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

---

## 🌐 Environment Variables

### Backend

Create a `.env`

```env
PORT=5000

MONGODB_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET
```
